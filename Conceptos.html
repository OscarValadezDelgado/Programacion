Tipos de malware

Resumen práctico para presentaciones y estudio
Tabla de tipos de malware con función e impacto asociado
Tipo de malware Función Impacto
Virus Se replica y adjunta a archivos o programas. Daño de datos, corrupción de archivos
Gusano (Worm) Se propaga por redes y sistemas sin intervención del usuario. Interrupción de red, consumo de recursos
Caballo de Troya Se disfraza de software legítimo para engañar al usuario. Robo de datos, control remoto del sistema
Ransomware Encripta o bloquea acceso a datos y exige rescate. Secuestro de datos, pérdida operativa
Spyware / Software espía Monitorea actividad y recopila información sin consentimiento. Violación de privacidad, robo de credenciales
Adware (Software publicitario) Inyecta o muestra anuncios no deseados; a veces rastrea hábitos. Molestias, degradación de rendimiento
Rootkit Permite ocultar malware y elevar privilegios en el sistema. Acceso no autorizado persistente
Keylogger Registra pulsaciones de teclado para robar contraseñas y datos. Robo de credenciales, filtración de información
Fileless malware (sin archivos) Se ejecuta en memoria utilizando procesos legítimos. Ataque sigiloso, difícil de detectar
Cryptojacker Utiliza recursos del sistema para minar criptomonedas. Alto uso de CPU/GPU, degradación del equipo
Botnet Red de dispositivos comprometidos controlados por un atacante. DDos, envío masivo de spam, ataques coordinados
Bomba lógica (Logic bomb) Ejecuta código malicioso al cumplirse cierta condición/fecha. Destrucción o modificación de datos
Wiper Elimina o sobrescribe datos de manera irreparable. Pérdida total de datos
Scareware Asusta al usuario con falsas alertas (p.ej. falso antivirus) para que pague o instale algo. Engaño, posibles pérdidas económicas
Consejo: Mantén el sistema actualizado, usa antivirus reputado y no ejecutes software desconocido.
